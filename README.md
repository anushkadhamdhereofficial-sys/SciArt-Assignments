# SciArt Assignment 2 — Scientific Infographic

## Objective
To create a scientific process figure based on a published 
research paper using BioRender.

## Paper Used
Title: The Ladder of Life Detection  
Authors: Marc Neveu et al.  
Journal: Astrobiology  
Year: 2018  

## Tool Used
BioRender (Basic Free Account)

## What the Infographic Shows
This infographic illustrates how scientists determine whether 
a detected compound in space (e.g. methane on Mars) is a 
genuine biosignature. It shows the two key criteria —
survivability and reliability — that must both be met before 
a compound can be considered evidence of life.
